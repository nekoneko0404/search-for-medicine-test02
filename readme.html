注意事項

  • 本ツールは、厚生労働省が公開しているExcelファイルに基づいて情報を表示します。データの正確性については、厚生労働省のウェブサイトで最新の情報を確認してください。
  • 本ツールは情報収集のサポートを目的としており、その正確性や完全性を保証するものではありません。ご利用によって生じるいかなる結果についても、責任を負いかねますのでご了承ください。
  • 本ツールの著作権は開発者に帰属します。無断での複製、配布、販売は禁止します。
  • PMDAのリダイレクトページ上の医薬品ガイド、RMP資材、電子化された添付文書等の著作権は製造販売業者が有します。無断での複製、転載、頒布は禁じられます。

使用方法

  • 検索条件の入力

    「医薬品名」「成分名」「規格、剤形、メーカー」の各項目に、検索したいキーワードを入力します。複数の項目にキーワードを入力することで、AND検索が可能です。各検索欄に複数のキーワードをスペース（全角・半角）区切りで入力すると、全てのキーワードが含まれる医薬品を絞り込むことができます。

  • 出荷状況での絞り込み

    「出荷状況」の欄にあるチェックボックスで、「通常出荷」「出荷制限」「供給停止」の表示/非表示を切り替えることができます。複数のチェックボックスにチェックを入れることで、表示したい状況を組み合わせることが可能です。

  • 検索の実行

    検索欄にキーワードを入力するだけで、自動的に検索結果が表示されます。リアルタイムで結果が更新されるため、スムーズにご利用いただけます。

  • 検索結果からの便利な操作

    検索結果の品名をクリックすると、PMDA（医薬品医療機器総合機構）の医薬品情報ページにアクセスできます。
    出荷状況をクリックすると、該当の医薬品のYJコードから詳細情報、類似薬検索ができるページにアクセスできます。
    また、成分名をクリックすると、検索欄がクリアされ、その成分名での検索が実行されます。

  • ソート

    「出荷状況」の項目をクリックすると、検索結果を「出荷状況」で並べ替えることができます。

このツールの目的と主な機能

  • 医薬品の出荷状況は、医療現場において非常に重要な情報です。厚生労働省は、この情報をExcelファイルとして公開していますが、そのデータ量は膨大であり、必要な情報を手作業で探し出すのは困難です。そこで、本ツールは、このExcelファイルを活用し、必要な情報を効率的に検索できるウェブアプリケーションとして開発されました。

  • 簡単な操作性

    自動データ更新: 本ツールは、厚生労働省が公開しているデータを自動的に更新します。ユーザーが手動でファイルをアップロードする必要はありません。

  • AND検索

    3つの検索項目: 「医薬品名」「成分名」「規格、剤形、メーカー」の3つの項目を組み合わせてAND検索できます。また、各検索欄に複数のキーワードをスペース区切りで入力して絞り込むことも可能です。

  • 文字の正規化

    自動変換: 全角・半角、ひらがな・カタカナ、大文字・小文字を自動的に統一します。

  • ソート機能

    出荷状況による並べ替え: 検索結果を「出荷状況」で並べ替えることができます。

  • レスポンシブデザイン

    PCとスマートフォンの両対応: PCだけでなく、スマートフォンでも快適に利用できるように設計されています。
